# PetSphere
A decentralized social platform for pet owners built on the Stacks blockchain. PetSphere allows users to:
- Create pet profiles
- Share posts about their pets
- Like and comment on other posts
- Follow other pet owners
- Earn reputation points through engagement

## Contracts
- `pet-profile.clar`: Handles pet profile creation and management
- `post.clar`: Manages post creation and interaction
- `social.clar`: Handles social features like following and reputation
